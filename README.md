# 🌑 NOCTURA – Alter Ego Digital

AI-powered self-reflection platform that creates a digital alter ego based on deep psychological profiling.

## ✨ Features

- 🔮 Deep Personality Scan (20-40 questions)
- 👤 AI Alter Ego Generator
- 💬 4 Conversation Modes (Wise, Brutal, Philosophical, Motivator)
- 🔮 Future Simulation Engine
- ✍️ Auto Writing Generator
- 🎮 Level & XP System

## 🛠️ Tech Stack

- Frontend: Vanilla JS + Tailwind CSS
- Backend: Node.js (coming soon)
- Database: PostgreSQL (planned)
